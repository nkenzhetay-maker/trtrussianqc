# TRT Russian — Система контроля качества

## Структура файлов
```
index.html                        → Интерфейс (аутентификация + редактор)
netlify/functions/quality-check.js → Claude API — анализ текста
netlify.toml                      → /api/* → /.netlify/functions/:splat
package.json                      → node-fetch
```

## Развёртывание на Netlify

1. Загрузить все файлы в GitHub репозиторий
2. Подключить репозиторий в Netlify
3. Добавить переменные среды в Netlify → Site settings → Environment variables:
   ```
   ANTHROPIC_API_KEY = sk-ant-...
   ```

## Как работает система

1. Редактор вставляет текст или загружает файл (.txt, .docx)
2. Выбирает тип контента: Новость / Видео / Инфографика / Статья / Соцсети
3. Нажимает "Проверить качество"
4. Система проверяет по 4 блокам:
   - Терминология (официальный глоссарий TRT Russian)
   - Позиция Турции / политика
   - Грамматика и орфография
   - Медийный стиль (недопустимые российские/западные нарративы)
5. Выносится вердикт: ОДОБРЕНО / ТРЕБУЕТ ПРАВОК / НЕ ОДОБРЕНО
6. Показывается исправленный текст + детальный отчёт по ошибкам
